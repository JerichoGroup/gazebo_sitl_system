## 📚 About

**gazebo_sitl_system** is a linux based simulation system for launching: 

- Gazebo (ignition) Garden.
- Ardupilot sitl.
- Mavros (ros2 humble).

## 📝 Build docker images
```shell
docker-compose build
```
## 🚀 Run simulation 
```shell
docker-compose up
```
